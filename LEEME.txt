Para crear el trustore:
- Exportar el certificado desde http://mail.sdd.com.ar
- Crear el ts con: /c/Program\ Files/Java/jdk1.6.0_45/bin/keytool.exe -import -alias mail_sdd_com_ar -file mail_sdd_com_ar.cer -keystore mail_sdd_com_ar.ts -storepass changeit


Nota de instalacion:
- Hacer un git clone https://github.com/eviera/Asignator.git
- En el Eclipse, importar con Existing Maven Projects
- En las propiedades del proyecto, agregar un target runtime (un jobss o tomcat) para que tenga las librerias servlet


Backend:
- Leer mails de soporte por imap
- Usar una blacklist ignorar algunos
- Por cada mail:
-- Llamar por REST a Jira para crear un ticket
-- Editar el subject del mail y pegarle el numero de ticket

Frontend:
- Llamar a Jira y traer todos los tkt no asignados
- Mostrar dos paneles, a la izquierda los tkt no asignados, a la derecha, los usuarios
- Soportar drag&drop de multiples tkts a un usuario
- El drop dispara una llamada REST a Jira para asignar el mail
- Los tkts asignados desaparecen del panel izquierdo
